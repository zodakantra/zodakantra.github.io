Proyectos

Acá presento algunos de mis proyectos más remarcables y creativos

Sistema de Gestión de Gimnasios

Un programa centrado en administrar los pagos, deudas y clientes de un gimnasio usando como almacenamiento de datos planillas de excel para fácil visualización, python con pandas para operaciones paralelizadas.

Lleva registro de deudores, cantidad y datos de todos los clientes además de los meses que va pagando cada uno, también gracias a su "modo registro" permite anotar los clientes que asisten día a día al establecimiento, logrando así llevar un atento seguimiento de los pagos.

Scrapper de OSINT

Un programa que se encarga de recopilar información de distintas fuentes de internet, como redes sociales, páginas web, foros, etc. para obtener información de una persona o entidad en particular.

Empleando selenium de python emula un navegador web para poder acceder a la información de las páginas web que requieran de un login o que tengan un sistema de seguridad que impida el acceso a bots. Guarda toda la información recaudada en un archivo de texto para su posterior análisis.

Bots de Whatsapp

Un bot diseñado para interactuar con WhatsApp y actuar como un navegador web mediante comandos específicos. Permite realizar diversas acciones como capturar pantallas, convertir páginas web a PDF y enviar el HTML de los enlaces proporcionados. Además, puede realizar búsquedas en Google y Twitter, devolviendo los resultados directamente en el chat de WhatsApp.

Factorizador de Polinomios Empleando el Método de Cardano

Utilizando el método de cardano en combinación con el de Ferrari este simple programa de consola permite factorizar polinomios de grado 3 y 4 obteniendo así sus raíces reales y complejas.

Teniendo en cuenta que el método de Cardano es un método general para resolver ecuaciones cúbicas, las raíces obtenidas están expresadas en términos reductibles pero muy complejos. Sería necesario un programa más avanzado para simplificar las raíces obtenidas.
